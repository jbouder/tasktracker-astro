## Why Tasker

<CardGrid stagger>
	<Card title="Typed end to end" icon="seti:typescript">
		React 19 + TypeScript on the front, async SQLAlchemy 2 + Pydantic on the
		back. The contract is typed at both ends.
	</Card>
	<Card title="On-brand by default" icon="puzzle">
		The UI is built from the **Nebari design system** — and so are these docs.
		One set of tokens themes the whole product.
	</Card>
	<Card title="Fast inner loop" icon="rocket">
		Vite + a FastAPI `--reload` server proxied at `/api`. Save a file, see the
		change.
	</Card>
	<Card title="Batteries included" icon="setting">
		Postgres, Alembic migrations, TanStack Query, and Jotai are wired up out of
		the box.
	</Card>
</CardGrid>
